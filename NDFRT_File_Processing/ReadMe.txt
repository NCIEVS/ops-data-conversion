This project attempts to access the Apelon website, download the NDFRT files, and process them.

1. Download from Apelon the 6 files.
2. Rename the files that need to be renamed - making dated copies of each
3. Unpack TDE xml file and put in it's own zip
4. Unpack NUI files and process
5. Unpack SPL files and process
5a.Create the three txt files with headers.
5b.Convert | to tab
5c.Create Excel files with headers.
5d.Load txt data into Excel and save
6. Unpack Release notes and process
7. Upload TDE-byName to FTP and notify Alameda
8. Upload All other files to public FTP
9. Notify Ops that files have been uploaded.
